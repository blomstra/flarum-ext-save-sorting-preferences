A [Flarum](http://flarum.org) extension. 

## Installation

Install with composer:

```sh
composer require blomstra/save-sorting-preferences:"*"
```

## Updating

```sh
composer update blomstra/save-sorting-preferences:"*"
php flarum migrate
php flarum cache:clear
```
